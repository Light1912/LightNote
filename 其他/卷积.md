离散卷积
$$
(f*g)(n) = \sum_{k=-\infty}^{\infty}f(k)g(n-k)
$$
https://www.zhihu.com/question/22298352/answer/34267457

理解为平移加权。



```python
fn = [1, 3, 2]
gn = [3, 2, 1]
fgn = [3, 9, 6, 0, 0] + 
      [0, 2, 6, 4, 0] + 
      [0, 0, 1, 3, 2]

# n = 0时 k只能取0
fgn = f(k) * g(0-k)
f(0) * g(0-0) = 1 * 3  = 3

# n = 1时, k取 0 和 1
fgn = f(k) * g(1-k)
    = f(0) * g(1) + f(1) * g(0)
    = 3 * 2 + 1 * 3
    = 9
```
