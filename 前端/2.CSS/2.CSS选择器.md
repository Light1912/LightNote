# CSS选择器

什么是选择器：用于选择需要添加样式的元素。

作用：找到应用样式的位置和范围。

**常用的选择器**：

- 标签选择器
- 类选择器
- 层级选择器(后代选择器)
- id选择器
- 组选择器
- 伪类选择器

## 标签选择器

以标签名为开头

所有的p标签文字都变为红色

```css
p {
    color: red;
}
```

## 类选择器

`.`+类名，比如

```css
.box {
    width:100px;
    height:100px;
    background:gold
}
```

## 多类名选择器

给标签指定多个类名，从而达到更多的选择目的。

注意：

- 样式显示效果跟HTML元素中的类名先后顺序没有关系,受CSS样式书写的上下顺序有关。
- 各个类名中间用空格隔开。

多类名选择器在后期布局比较复杂的情况下，还是较多使用的。

```html
<div class="blue fontWeight font20">李白</div>
<div class="font20">杜甫</div>
<div class="font14 gold">白居易</div>
<div class="font14">王维</div>
```

## id选择器

`#`+id名。一般id只对应一个元素，所以不推荐用id作为选择器。一般用类选择器就行了。

```css
#box {
    width:100px;
    height:100px;
    background:gold
}
```

## 通配符选择器

匹配所有元素

```css
* { 
    margin: 0; /* 定义外边距*/ 
    padding: 0; /* 定义内边距*/ 
}
```

## 伪类选择器

伪类选择器用于向某些选择器添加特殊的效果。比如给链接添加特殊效果；选择第1个或第n个元素。

核心特点——分号`:`

### 链接状态效果

```css
.box {
    width:100px;
    height:100px;
    background:gold
}

.box:hover{
    background:lightyellow
}
```

鼠标悬停在这个元素上时变色

其他效果：

- `link`，未访问
- `visited`，已访问
- `hover`，鼠标悬停
- `active`，鼠标按下未抬起

### 位置伪类选择器

- `:first-child` :选取属于其父元素的首个子元素的指定选择器
- `:last-child` :选取属于其父元素的最后一个子元素的指定选择器
- `:nth-child(n)` ： 匹配属于其父元素的第 N 个子元素，不论元素的类型
- `:nth-last-child(n)` ：选择器匹配属于其元素的第 N 个子元素的每个元素，不论元素的类型，从最后一个子元素开始计数。

n 可以是数字、关键词或公式

作为第一个子元素的`li`元素。也就是说，如果一个标签有3个元素，而第一个元素不是`li`，那么就不会改变样式。

无视嵌套：多层嵌套，只要li标签是第一个子元素，就会生效。



```css
li:first-child {
    /* 选择第一个子元素 */ 
    color: gold; 
}

li:last-child { 
    /* 最后一个子元素 */ 
    color: green; 
}

li:nth-child(4) { 
    /* 选择第4个子元素 */ 
    color: skyblue; 
}

li:nth-last-child(4) { 
    /* 选择倒数第4个子元素 */ 
    color: skyblue; 
}

li:nth-child(odd) { 
    /* 选择奇数元素 */ 
    color: skyblue; 
}

li:nth-child(even) { 
    /* 选择偶数元素 */ 
    color: skyblue; 
}

li:nth-child(3n) { 
    /* 选择3的倍数元素 */ 
    color: skyblue; 
}

span:nth-of-type(2) {
    background-color: orange;
}

span:last-of-type(2) {
    background-color: orange;
}

span:only-of-type {
    background-color: orange;
}

span:only-child {
    background-color: orange;
}

/* 根元素选择器，等同于选中了html元素 */
:root {
    background-color: orange;
}

div:empty {
    width: 50px;
    height: 50px;
    background-color: green;
}
```

- `span:nth-of-type(2)`：一个元素可能有很多个子标签，我只想改变第二个`span`标签的样式，就可以用这个方式
- `span:only-of-type`：元素有多个子标签，但只有一个span标签，就改变它的样式
- `span:only-child`：元素只有一个子标签且是span标签时，改变它的样式
- `div:empty`：div标签中如果**没有任何内容**，则改变样式

### 目标伪类选择器

选择器可用于选取当前活动的目标元素

```css
:target 
{ 
    color: red; 
    font-size: 30px; 
}
```

结合锚点使用，点击跳转到某个标题时，该标题就会改变样式。

> 目标伪类选择器是一个动态选择器，只有存在url指向该匹配元素时，样式才生效。

## 复合选择器

就是之前学过的选择器组合起来使用

### 交集选择器

由标签选择器与class选择器构成，中间不能有空格。

使用较少，不建议用

```css
p.box { 
    background-color: #32CD32;
}
```

### 并集选择器

以逗号`,`分割，任何形式的的选择器都行

```css
p,
.box {
    width:100px;
    background:gold;
}
```

### 后代选择器

用于选择元素或元素组的后代。

用空格分隔。

```css
.box span {
    background:gold
}
```

对应的html案例

```html
<div class="box">
    <span>这里是span标签</span>
    <p>
        这里是p标签
    </p>
</div>
```

box类标签的子元素如果是`span`标签，那么就应用对应的样式。

> 层级关系包含所有后代，不仅是子标签，孙子标签也会有效

### 子元素选择器

和后代选择器不同，只选中子代。

使用`>`进行连接。

```css
.box > span {
    background:gold
}
```

### 相邻元素选择器

选择紧跟在另一元素后的元素，且二者有相同的父元素。

```css
h1 + p {
    background:gold
}
```

### 兄弟选择器

作用于某一个指定元素后面的所有兄弟结点。

```css
.box ~ a {
    color:blue
}
```

类box之后的所有a标签兄弟。

### 属性选择器

```css
/* 有type属性的标签 */
input[type] {
    background:red
}

/* class属性值以color开头 */
p[class^=color] {
    background:red
}

/* class属性值以color结尾 */
p[class&=color] {
    background:red
}

/* class属性值包含color */
p[class*=color] {
    background:red
}
```

ps:emmit语法

```html
<!-- p[class=book]{$}*3 -->
<p class="book">1</p>
<p class="book">2</p>
<p class="book">3</p>
```

### 伪元素选择器

- 选择文本的第一个字（中、日、韩）
- 文本第一行
- 可改变选中文本的样式，按住鼠标拖动选中文字。

```css
/* 文本的第一个字 */
p::first-letter {
    font-size: 20px;
    color: blue;
}

/* 第一行的样式 */
p::first-line {
    font-size: 20px;
    color: blue;
}

/* 选中文本的样式 */
p::selection {
    font-size: 20px;
    color: blue;
}
```

在E元素内部的开始位置和结束位创建一个元素，该元素为行内元素，且必须要结合content属性使用。

>E:after、E:before 在旧版本里是伪元素，CSS3的规范里“:”用来表示伪类，“::”用来表示伪元素，但是在高版本浏览器下E:after、E:before会被自动识别为E::after、E::before，这样做的目的是用来做兼容处理。

```css
h3::before {
    content: "《";
}

h3::after {
    content: "》";
}
```
