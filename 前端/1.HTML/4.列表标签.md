# 列表标签

## 无序列表

- `<ul>` unordered lists
- `<li>`list item

```html
<ul>
    <!-- li标签定义列表项目 --> 
    <li>列表标题一</li>
    <li>列表标题二</li>
    <li>列表标题三</li>
</ul>
```

## 有序列表

- `<ol>`：ordered list

```html
<ol>
    <!-- li标签定义列表项目 --> 
    <li>列表标题一</li>
    <li>列表标题二</li>
    <li>列表标题三</li>
</ol>
```

## 定义列表

- `<dl>`：definition list
- `<dt>`：definition term，列表项的标题
- `<dd>`：definition description，列表项的内容

```html
<dl>
    <dt>标题1</dt>
    <dd>内容1</dd>
    <dd>内容2</dd>
</dl>
```

## emmet快速书写

输入后按TAB

```
ul>li*3

ol>li*3

dl>dd*3
```
