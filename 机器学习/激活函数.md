Sigmoid型函数

特指一类S型曲线函数，在x趋近于正负无穷时，两端趋近于一个固定的值。

|                                                              |      |      |
| ------------------------------------------------------------ | ---- | ---- |
|                                                              |      |      |
| $$\sigma(x) = 1 / (1+\text{exp}(-x))$$                       |      |      |
| $$\tanh(x) = (\text{exp}(x) - \text{exp}(-x)) / (\text{exp}(x) + \text{exp}(-x))$$ |      |      |


$$
\sigma(x) = \frac{1}{1-e^{-x}}
$$
