# python2和python3的区别

|                | python2                                                      | pyhon3           |
| -------------- | ------------------------------------------------------------ | ---------------- |
| 经典类和新式类 | 不指定父类则为经典类                                         | 不存在经典类     |
| 中文的支持     | 需要加入`#-*- coding:utf8 -*-`                               | 默认支持中文     |
| print语句      | `print 'hello'`                                              | `print('hello')` |
| input语句      | 会把用户的输入当做代码<br />`raw_input()`就跟python3的`input()`一样 | 输入的是字符串   |
