自己创建一个字典类，实现以访问属性的形式访问字典值

```python
class MyDict(dict):
    def __init__(self, d):
        super().__init__()
        self.update(d)
    
    def __getattr__(self, key):
        return self[key]
    
    def __setattr__(self, key, value):
        self.__dict__[key] = value
        
d = MyDict({'a':2, 'b':3})
print(d)
print(d.a)
print(d.b)
```

{'a': 2, 'b': 3}
2
3
