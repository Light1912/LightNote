## 画矩形框

```python
import numpy as np
import matplotlib.pyplot as plt

fig, ax = plt.subplots()
x = np.linspace(0, 20, 1000)
y = np.sin(x)
rect = plt.Rectangle((.25, .25), .5, .5, transform=ax.transAxes, ec='r', fc='None')
ax.add_patch(rect)
plt.plot(x, y)
```
