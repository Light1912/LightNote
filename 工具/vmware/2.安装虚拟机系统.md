# CentOS 8.5

https://blog.csdn.net/qq_41951895/article/details/121984630

https://blog.csdn.net/qq_34146694/article/details/104475916

安装过程中，选择最小安装即可。

删除其他源，使用阿里源作为下载源。

```
cd /etc/yum.repos.d/
rm -rf *
curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-8.repo
yum makecache
```
