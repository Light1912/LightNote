## 元数据

| 名字                | 解释                                                         | 举例                       |
| ------------------- | ------------------------------------------------------------ | -------------------------- |
| verbose_name        | 用于设置模型对象的直观、人类可读的名称，用于在各种打印、页面展示等场景。 | `verbose_name = "story"`   |
| verbose_name_plural | 复数名                                                       | `verbose_name = "stories"` |
| ordering            | 默认升序排列<br />“-”则表示按降序排列<br />问号“？”表示随机排列 | `ordering = ['-pub_date']` |
|                     |                                                              |                            |
